**Preciso de contexto rápido**

## 📋 Análise de Confiança

A pasta `Níveis de Confiança/` contém:
- Análises de confiança do sistema
- Gráficos de evolução
- Métricas de desempenho
- Script de geração de gráficos em `grafo/grafo.py`

Para regenerar gráficos:
```bash
python docs/models/Niveis\ de\ Confiança/grafo/grafo.py
```
